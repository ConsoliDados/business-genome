# Skill — capture-idea

> **CLI:** run when the human says *"I have an idea"*. **Web-chat:** paste the prompt.

**Writes:** `12-editorial/ideas/<slug>.md` (template: `96-templates/tp-idea.md`)

## Steps
1. Capture: the idea (1–2 lines), why it matters / the angle.
2. **If the channel isn't stated, ASK** — blog / LinkedIn / Substack / X / Instagram /
   youtube-script. Set `channel` once known.
3. Link to a `series` if it belongs to a content line.

## Guard-rails
- Never assume the channel. Don't over-build a raw idea — that's [[draft-post]]'s job.

---
## Prompt (copy-paste)
> I have a content idea: <idea>. If I didn't say the channel, ask me which (blog/LinkedIn/
> Substack/X/Instagram/YouTube). Capture it as a short idea note with the angle and possible
> channels.
