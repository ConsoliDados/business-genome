# Skill — discovery

> **CLI:** run to structure a Discovery Pack. **Web-chat:** paste the prompt.

**Writes:** `…/commercial/discovery.md`

## Steps
1. Pick tier: **Light / Standard / Deep** (Standard is the most-sold). Process:
   **NDA → kickoff → execution → presentation**.
2. Produce: scope studied, proposed architecture (high level), justified stack, risks, phased
   roadmap + estimate.
3. Output feeds the [[proposal]]. Discovery is **100% credited** to the project's first invoice.

## Guard-rails
- Deep is mandatory before pricing a T4 modernization.

---
## Prompt (copy-paste)
> Draft a Discovery Pack (tier: Light/Standard/Deep) for this engagement: scope, high-level
> architecture, justified stack, risks, and a phased roadmap with a ranged estimate. End with a
> recommendation that feeds a 3-package proposal.
