# Skill — portfolio-review

> **CLI:** run for a weekly/portfolio roll-up across projects. **Web-chat:** paste the prompt.

**Reads:** `00-dashboards/` + `11-projects/` + `10-crm/` · **Writes:** a dated note in `14-meeting-notes/`
or `15-daily/`.

## Steps
1. Pipeline: leads → discovery → projects (conversion).
2. Active projects by type/status; flags (stalled, at-risk).
3. Finance: margin-vs-target per active project, monthly run-rate, FX exposure (BRL T3+).
4. Editorial: what's scheduled/published this week.
5. Output: 5–8 bullets + the top 3 decisions the human needs to make.

---
## Prompt (copy-paste)
> Give me a portfolio review: pipeline conversion, active projects by status with risk flags,
> margin-vs-target and run-rate, editorial scheduled this week, and the top 3 decisions I need
> to make now.
