# Skill — marketing-plan

> Chapter of [[README|business-plan]]. The **go-to-market** side (distinct from [[market-study]],
> which is the *analysis*). **CLI:** run for the marketing chapter. **Web-chat:** paste.

**Writes:** the *Marketing plan* section of `…/strategy/business-plan.md`
**Reuses:** [[market-study]] (segments/positioning) · [[persona-study]] (who)

## Steps
1. **Products & services** description (what's included per offering).
2. **Pricing model** + what drives it (value-based / dynamic / commission); guardrails.
3. **Value proposition** mapped to each persona's pains.
4. **Promotional strategy** — channels, campaigns, triggers.
5. **Sales / commercialization structure** — motion, reps, channels.
6. **GTM** — inbound/content/direct; sequencing.

## Guard-rails
- Pull personas/positioning from their skills — don't re-derive. Tie every tactic to a persona.
