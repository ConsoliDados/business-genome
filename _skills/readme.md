# Skill — readme

> **CLI:** run to (re)generate a project's `README.md` home. **Web-chat:** paste the prompt.

**Writes:** `projects/<area>/<slug>/README.md`

## Steps
1. Keep the **declarations block** current (project-type, doc-tier, engagement-tier, currency,
   language, `client:`, `repo:`).
2. One-liner + a module index with wikilinks to the files that exist for this type.

## Guard-rails
- Someone should read this and understand/run the project without asking you.

---
## Prompt (copy-paste)
> Generate the project README: declarations block, one-line summary, and a linked index of the
> project's modules.
